This is Bootiful Micro Pizza. A fictitious online pizza store built by Ian Collington - a UK based Java consultant.

It has been created to explore and learn the technology and architecture around microservices. This is a work in progress so there is limited functionality at this time.

This site is being developed using the latest technologies. On the backend we have:-

  • Spring Boot 2.1.2
  • Spring Cloud Greenwich Release
  • MySQL (RDS)
  • Java 8

On the frontend (this website) we have:-

  • Angular 7
  • NGINX

It is hosted on AWS using:-

  • Elastic Container Service for Kubernetes (EKS)
  • Elastic Container Registry (ECR)
  • S3
  • Relational Database Service (RDS)
  • Route 53

Built using the following tools:-

  • GitHub
  • AWS CloudFormation
  • AWS CodePipeline
  • Maven
  • Docker

You can find out more about this project on Ian's blog.
